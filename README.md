# EEG-fMRI Workshop 2026 Website

Static public website for the EEG-fMRI Workshop 2026 in Athens, Greece.

## Contents

- `index.html` - main page
- `css/` - site styles
- `js/` - site behavior
- `images/` - public site images and logos
- `data/interest-summary.json` - aggregate, non-identifying interest summary
- `CNAME` - GitHub Pages custom domain (`eegfmri.org`)

## Deployment

This repository is intended for GitHub Pages. Configure Pages to publish from the
repository root on the default branch, and keep the custom domain set to
`eegfmri.org`.

Do not commit raw registration exports, Google Form response CSV files, private
workflow notes, or local assistant settings.
